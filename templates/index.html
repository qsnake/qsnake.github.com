About

Qsnake is a source distribution for scientific computing, with unified Python interface.

Download and Install

Follow the instructions in the README.

Supported platforms: Linux, Mac (OSX). If something doesn't work, please report it to the main project mailinglist or issues.

Features

The standard packages include:

  • IPython
  • NumPy
  • SciPy
  • SymPy
  • Matplotlib
  • Qsnake Standard Library

Development tools include:

  • CMake
  • Cython
  • HDF5 (h5py)
  • Blas, Lapack
  • pysparse

Optional modules include:

  • Electronic structure calculation codes
  • Finite element libraries
